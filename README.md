# Rock Paper Scissors Game

A classic Rock Paper Scissors game built with vanilla JavaScript. Play against the computer and track your score in real-time.

## Features

✨ **Core Gameplay**
- Choose rock, paper, or scissors
- Computer generates random choice
- Real-time win/loss/draw detection
- Live score tracking for both player and computer

🎨 **UI/UX**
- Color-coded feedback messages
  - 🟢 Green: You win
  - 🔴 Red: You lose
  - 🔵 Blue: Draw
- Clean, responsive interface

## Tech Stack

- **Language:** Vanilla JavaScript
- **DOM:** Query selectors & event listeners
- **Styling:** CSS

## How to Play

1. Click rock, paper, or scissors
2. Computer makes its choice instantly
3. Win/loss is displayed with your current score
4. Play as many rounds as you want!

## Game Logic

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
